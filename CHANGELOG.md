## 1.0.0 (2025-02-16)

### 功能
- 基于 MetingAPI 的点歌插件
- 支持自定义 MetingAPI 地址
- 支持QQ音乐、网易云、酷狗、酷我音源
- 支持切换当前会话的音源
- `点歌 <关键词>` 搜索歌曲
- `点歌x`（不带空格）播放第x首搜索结果
- 歌曲自动分段为语音消息发送

### 安全性
- URL 安全校验，防止 SSRF 攻击
- API URL 安全校验
- 文件头魔数检测，验证音频文件有效性
- 临时文件使用唯一命名空间前缀

### 并发安全
- 会话数据封装为 SessionData 类，避免裸 dict 外泄
- 会话清理操作在锁保护下进行
- 临时文件追踪和清理机制
